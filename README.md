# project_data

A small repository to store project-related data and resources.

## Contents

- `data/` - Raw and processed datasets
- `scripts/` - Utility scripts for data processing and analysis
- `docs/` - Documentation and notes

## Usage

1. Clone the repository:

```bash
git clone https://github.com/mccreation00-ctrl/project_data.git
```

2. Explore the `data/` and `scripts/` folders. Follow any README files inside subdirectories for details.
